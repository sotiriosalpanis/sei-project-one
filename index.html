Tetris

How to Play

  • Use the arrow keys to move left or right
  • Rotate the active piece using the up key
  • The down key will increase the drop speed
  • Points are scored when a row is completed
  • Clearing multiple rows will score bonus points
  • The game will become more difficult as you score more points

Good Luck and enjoy!
